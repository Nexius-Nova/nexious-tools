``` 2. 数据库层优化 ( db-sqlite.js )
问题：

- 数据库迁移逻辑与初始化逻辑混在一起（第 242-512 行）
- 缺少版本化迁移系统
- 每次启动都检查表结构，效率较低
建议：

```

const migrations = [
  { version: 1, up: 
  createInitialTables },
  { version: 2, up: addImagesColumn },
  { version: 3, up: 
  addUniqueConstraints },
];

async function runMigrations() {
  const currentVersion = await 
  getDbVersion();
  for (const migration of migrations) 
  {
    if (migration.version > 
    currentVersion) {
      await migration.up();
      await setDbVersion(migration.
      version);
    }
  }
}
